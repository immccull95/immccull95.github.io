Matthew Moore GIS Portfolio

Utah Mountain Lions Movement Patterns

This project is the results of working with a dataset that

tracked two different mountain lions in Utah named F53 and F64.

This map shows things like the Central Features of both the F53

and the F64 mountain lion.This map also shows how compact and dispersed

the movement patterns for the mountain lions along with the directional trends

of both mountain lions. A bunch of different tools were used for the final output including

Standard Distance, Mean Center, Directional Distribution (SDE) and Central Feature.

Watersheds In Upstate New York

This map shows the output of mapping

and working with watershed dilineation in Adirondack Park

in upstate New York. Using different tools like the Con,

Flow Direction, Sink, Fill, Watershed along with many others,

we were able to map out the water sheds for the study area

of Adirondack Park in upstate New York.

Mountain Goat Range Weighted Raster Comparison

This project is the output of working with and comparing two different

raster sets regarding a Mountain Goat Range in Colorado. This project compared a DEM

(Digital Elevation Model) and a NLCD (National Land Cover Database) raster of the same

Mountain Goat Range. Using the Slope tool to find the slope and Reclassify to change the

values in a raster, we were able to compare the model weight of the slope, NLCD, and DEM

models.
